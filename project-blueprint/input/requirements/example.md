# 社内の備品管理をなんとかしたい

> これはサンプルです。要求メモを書くときの参考にしてください。
> 書き方は自由です。箇条書きでも会議メモの貼り付けでもOKです。

## 背景

- 総務部がExcel台帳で備品を管理しているが、運用が回らなくなってきた
- 毎月の棚卸しで丸一日取られ、本来業務を圧迫している
- 来期（4月〜）までに改善したいという話が出ている

## 困っていること

- Excelで備品の在庫を管理しているが、誰が何を持っているかわからなくなる
- 新しい備品を買ったときに台帳に書き忘れる人が多い
- 棚卸しのたびに丸一日かかっている
- 壊れた備品がそのまま放置されていることがある

## こんな感じにしたい

- スマホからでも「この備品を借ります」と登録できる
- 誰が何を持っているか一覧で見たい
- 備品が壊れたら報告できて、管理者に通知が飛ぶ
- 棚卸しのときにチェックリストが出てくると嬉しい
- 部署ごとに「うちの部署でどのくらい備品使ってるか」がわかるといい

## 使う人

- 一般社員（200人くらい）: 備品を借りたり返したりする
- 総務の管理者（3人）: 備品の登録、廃棄、棚卸し
- 部門長（10人くらい）: 自部署の利用状況を確認

## 制約・条件

- 社内ネットワークからのアクセスだけでOK（社外からは不要）
- 既存のExcel台帳のデータは移行したい（500件くらい）
- 予算はあまりない。できれば既存のサーバーで動かしたい
- 来期（4月）の棚卸しまでに、少なくとも貸出管理は動いてほしい

## まだ決めていないこと

- バーコードやQRコードで備品を読み取れると楽かも？ でも必須ではない
- スマホアプリが要るか、ブラウザで十分か
- 貸出時に上長の承認フローが必要か

## 優先度

1. まずは貸出・返却の管理ができること（これが一番困ってる）
2. 次に備品台帳の管理（登録・編集・廃棄）
3. 棚卸し機能はあとでもいい
4. レポート・集計は余裕があれば
